# Изменения в ЕГРЮЛ/ЕГРИП
Метод позволяет получить данные об изменениях вносимых в ЕГРЮЛ/ЕГРИП.

## Параметры запроса

Для получения сведений связанных с изменениями вносимыми в реестры, необходим запрос `records`.



## Пример запроса
``` Graphql
query {
  company(ogrn: "1117746390277"){
    records{
      grn
      grnDate
      type
      authority{
        code
        name
      }
      documents{
        number
        name
        date
      }
    }
  }
}
```
