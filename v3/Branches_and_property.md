# Филиалы и дочерние компании
Метод позволяет получить информацию о филиалах и дочерках компании.

## 1. Параметры запроса

Для получения информации о филиалах необходим запрос `branches`.


## Пример запроса
``` graphql
query {
  company(ogrn: "1028601792878"){
    fullName
    branches{
      fullName
      grnDate
      address{
        country
        full
        region{
          full
          code
          name
        }
        house
      }
      officeType
    }
  }
}
```

## 2. Параметры запроса
Для получения информации о дочерних компаниях необходим запрос `property`.

## Пример запроса
``` graphql
query {
  company(ogrn: "1023404362662"){
    fullName
    property{
      share
      amount
      grnDate
      property{
        fullName
        ogrn
        inn
      }
    }
  }
}
```
