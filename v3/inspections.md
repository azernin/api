# Проверки
Метод позволяет получить детализированную информацию проверках компании проводимых органами государственной власти.

## Параметры запроса

Для получения информации о проверках необходим запрос `inspections` внутри выбранной компании.

Должен быть задан обязательный фильтр: `first`.

* ``(first: 3)`` -фильтр для получения информации по последним проверкам, где число указывает на количество запрошенных последних проверок у компании.
* ``(After:cursor)`` - инструмент для пагинации проверок.

## Пример запроса

```Graphql
query company {
  company(ogrn: "1026901668837") {
    ogrn
    fullName
    inspections{
      list(first: 3) {
        id
        erpId
        status
        type
        schedule
        startDate
        endDate
        purpose
        reasons {
          title
          text
        }
        objects {
          type
          address
          violations {
            type
            text
            injunctions {
              number
              isViolated
              creationDate
              deadlineDate
            }
          }
        }
        authorities {
          ogrn
          fullName
        }
      }
      statistics {
        totalCount
      }
    }
  }
}
```
