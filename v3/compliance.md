# Факторы риска
Метод позволяет получить информацию о рисках работы с контрагентом. В том числе, рекомендованных ФНС для проявления должной осмотрительности.

## Параметры запроса

Для получения информации о Рисках необходим запрос `compliance`.

На данный момент система Рисков претерпевает изменения, поэтому основная часть рисков расположены в поле `risks`, часть в поле `markers` .


## Пример запроса
```graphql query company {
  company(ogrn: "1117746390277") {
    ogrn
    fullName
    compliance {
      score
      certainty
      risks {
        risk
        title
        description
        source
        occurrences {
          severity
          relation
          entity {
            ogrn
            fullName
          }
          description {
            ... on DescriptionPerson {
              position
              person {
                firstName
                middleName
                lastName
                inn
              }
            }
            ... on DescriptionCompany {
              company {
                ogrn
                fullName
              }
            }
          }
          date
          entry
          age
          count
          amount
          share
        }
      }
    }
  }
}
```

## Новая система рисков
`markers`. На данный момент, в новой системе рисков есть только информация о Блокировке расчетных счетов.

## Пример запроса

``` graphql query company {
  company(ogrn: "1117746390277") {
    ogrn
    fullName
    compliance {
      score
      certainty
      markers {
        code
        name
        source {
          name
          date
        }
        occurrences {
          title
          description
          severity
          weight
          date
        }
      }
    }
  }
}
```
